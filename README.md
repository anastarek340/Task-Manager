# Task Manager

Simple **CRUD web application** built using **PHP & MySQL** to manage tasks.

## Features

* Add new tasks
* View all tasks
* Update existing tasks
* Delete tasks

## Technologies Used

* PHP
* MySQL
* HTML
* CSS

## Project Purpose

This project was created to practice **CRUD operations** and **database connection using PHP and MySQL**.
